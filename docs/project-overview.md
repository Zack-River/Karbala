# Project Overview

## Project Name

وعيٌ يمرّ من كربلاء

## Project Type

Educational Islamic Content Platform

## Purpose

Create a premium Arabic platform that presents a knowledge season consisting of 13 nights.

The platform should allow visitors to:

* Browse the season
* Explore all nights
* Read detailed content
* Access resources
* Listen to audio recordings
* View and download cards
* Share content

The platform should allow administrators to:

* Manage season information
* Manage nights
* Manage cards
* Manage resources
* Manage attachments
* Upload media files

## Public Pages

* /karbala
* /karbala/night/[slug]
* /karbala/cards
* /karbala/cards/[slug]

## Administration

* /admin

## Core Requirements

* Arabic First
* RTL First
* Mobile First
* Responsive Design
* SEO Friendly
* Accessibility Friendly
* Dynamic Content
* Production Ready

# Agent Rules

Before writing any code:

* Read all markdown files completely.
* Understand the design system before implementation.

Development Rules:

* Use Next.js App Router.
* Use TypeScript.
* Use Tailwind CSS.
* Use Supabase.
* Use reusable components.
* Follow RTL conventions.
* Follow mobile-first principles.

Content Rules:

* No hardcoded content.
* No lorem ipsum.
* No placeholder business content.
* Use database as the source of truth.

Design Rules:

* Follow design-system.md exactly.
* Do not redesign components.
* Do not invent new visual styles.
* Maintain visual consistency.

Quality Rules:

* Production Ready
* Responsive
* Accessible
* SEO Friendly
* Optimized
* Clean Architecture
